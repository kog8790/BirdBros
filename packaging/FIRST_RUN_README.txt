BirdBros Recycle Co. — First Run

Install:
1. Drag BirdBros Recycle Co. into Applications.
2. Open it from Applications.

macOS permissions:
The app watches a selected screen region, so macOS may ask for Screen Recording permission.
If the capture window is black or frozen:
1. Open System Settings.
2. Go to Privacy & Security.
3. Open Screen Recording.
4. Enable BirdBros Recycle Co.
5. Quit and reopen the app.

OpenAI key:
BirdBros needs an OpenAI API key for AI behavior judgment.
For a non-technical public release, the app should include a first-run API key setup screen or use a hosted API backend.
Do not publish a build with your personal API key embedded.
